# 15.1.1 启动管理-启动流程-运行级别

## 运行级别

| 运行级别 | 含义 |
| ---- | ---- |
| 0 | 关机 |
| 1 | 单用户模式，可以相像为windows的安全模式，主要用于系统修复 |
| 2 | 不完全命令模式，不含NFS服务 |
| 3 | 完全的命令模式，就是标准字符界面 |
| 4 | 系统保留 |
| 5 | 图形模式 |
| 6 | 重启动 |

## 运行级别命令

```bash
# 查看运行级别命令
runlevel

# 改变运行级别命令
init 运行级别
```
## 系统默认运行级别

```bash
vim /etc/inittab
# 系统开机后直接进入哪个运行级别
id:3:initdefault:
```
